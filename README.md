# P.A.S
Projeto Alimentar &amp; Saber (P.A.S.): plataforma educativa para incentivar hábitos alimentares saudáveis e desperdício zero. Oferece conteúdo multimídia e ferramentas interativas, como a Calculadora de Desperdício, com foco em sustentabilidade e conscientização social. Tecnologias utilizadas: HTML, CSS, JavaScript, API


# Como rodar o projeto
Clone o repositório e execute os comandos abaixo:

## Instalar dependências
npm install

## Rodar o projeto em ambiente de desenvolvimento
npm run dev


A aplicação estará disponível em:
http://localhost:5173/
